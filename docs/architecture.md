# Infrastructure Scripts Architecture

## Overview

This repository provides infrastructure administration, troubleshooting, monitoring, and diagnostic scripts commonly used by systems administrators and infrastructure engineers.

## Functional Areas

### Network Diagnostics

- Test-NetworkConnectivity.ps1
- Test-PortConnectivity.ps1
- Test-DnsResolution.ps1

### Infrastructure Reporting

- Get-NetworkInfo.ps1
- Get-NetworkConfiguration.ps1

### Troubleshooting

- Get-SystemErrors.ps1
- Get-RunningServices.ps1
- Restart-NetworkAdapter.ps1

### Performance Monitoring

- Get-TopProcesses.ps1

## Workflow

Infrastructure Issue
        ↓
Network Diagnostics
        ↓
System Analysis
        ↓
Reporting
        ↓
Troubleshooting
        ↓
Resolution
